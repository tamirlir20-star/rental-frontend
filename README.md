# מחירון שכירות ישראל

פלטפורמת השוואת מחירי שכירות בערים ושכונות ישראל.

## תכונות

- **חיפוש וסינון** — בחירת עיר ושכונה, מספר חדרים, טווח מחיר, ונוחיות (חניה, מרפסת, מעלית, חיות מחמד, מרוהטת)
- **תוצאות** — כרטיסיות עם מחיר ממוצע, מחיר למ"ר, חציון, מגמה ושינוי שנתי. עם תג "ערך מעולה" לשכונה המשתלמת ביותר
- **השוואה** — טבלת השוואה ל-2–4 שכונות עם הדגשת הערך הנמוך ביותר בכל עמודה
- **גרפים** — גרף קו של מחיר ממוצע לאורך 12 חודשים, גרף עמודות של מחיר למ"ר לפי עיר
- **מפה** — מפת ישראל עם עיגולים צבעוניים לפי רמת המחיר, לחיצה לפרטים

## טכנולוגיות

- React 19 + Vite 8
- TypeScript
- Tailwind CSS v4
- Recharts (גרפים)
- React Leaflet (מפה)
- Zustand (ניהול state)

## התקנה והרצה

```bash
# התקנת dependencies
npm install

# הפעלת שרת פיתוח
npm run dev

# בניית גרסת production
npm run build
```

האפליקציה זמינה בכתובת: http://localhost:5173

## מבנה הקוד

```
src/
  components/
    FilterPanel.tsx      — לוח סינון (עיר, שכונה, חדרים, מחיר, נוחיות)
    ResultsGrid.tsx      — רשת תוצאות עם skeleton loading
    PropertyCard.tsx     — כרטיסיית שכונה בודדת
    ComparisonTable.tsx  — טבלת השוואה צד-לצד
    RentChart.tsx        — גרפי Recharts (קו + עמודות)
    RentMap.tsx          — מפת React Leaflet
  data/
    rentals.ts           — נתוני מחירים ריאליסטיים ל-8 ערים ו-3–5 שכונות כל אחת
  store/
    useFilterStore.ts    — Zustand store לניהול כל מצב הסינון
  App.tsx                — קומפוננטת ראשית עם navigation
  main.tsx
```

## נתונים

הנתונים מייצגים נתוני שוק ישראלי 2024–2025 עבור הערים:
תל אביב, ירושלים, חיפה, באר שבע, ראשון לציון, נתניה, אשדוד, פתח תקווה.

הנתונים הם לצורכי הדגמה בלבד ואינם מחייבים.
